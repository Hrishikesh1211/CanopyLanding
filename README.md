# Canopy Landing Page

A modern, responsive landing page for Canopy.

## Overview

This project contains the HTML and styling for the Canopy landing page, designed to be clean, professional, and user-friendly.

## Files

- `canopy-landing.html` - Main landing page with integrated styling

## Features

- Responsive design
- Modern layout
- Professional appearance

## Getting Started

Simply open `canopy-landing.html` in a web browser to view the landing page.

## Development

To customize the landing page, edit the HTML file directly. All styling is included within the HTML document.
